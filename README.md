# Replicator Dynamics

Evolutionary game theory: replicator equations, ESS analysis, Moran process, and simplex phase portraits.

## Quickstart

```bash
git clone https://github.com/abhi-wadhwa/replicator-dynamics.git
cd replicator-dynamics
make install
make run
make test
```
